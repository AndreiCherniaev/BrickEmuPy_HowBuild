# BrickEmuPy
Brick Game (aka Tetris) emulator in Python with PyQt6.<br/>
Currently only the E-23 PLUS MARK II 96 in 1 model is emulated.

An [article](https://habr.com/ru/articles/773040/) describing the reverse engineering.
